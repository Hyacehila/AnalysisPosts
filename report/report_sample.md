# 郑州大学生夜骑开封事件舆情分析报告

## 1. 报告摘要

基于对2024年6月1日至11月30日期间17,275条社交媒体数据的综合分析，本次舆情事件整体呈现"中性为主，正面略多于负面"的特征。情感分布数据显示，中性情感博文占比46.6%（8,049条），正面情感博文占比28.5%（4,931条），负面情感博文占比23.9%（4,123条）。事件核心围绕"郑州大学生夜骑开封"展开，在2024年11月9日达到舆情高峰，单小时发帖量达507条，远超平均水平（23.79条/小时）。

**关键发现**：
- **舆情焦点**：民众反应(28.31%)、交通运输(20.36%)和社会民生(14.97%)为三大核心主题
- **地域集中**：河南省发帖量占比32.38%（5,593条），为舆情最活跃区域
- **发布者多元**：个人用户(62.07%)和自媒体(18.13%)为主要发声渠道
- **传播特征**：总互动量达521.6万次，其中转发116.1万次，评论42.7万次，点赞362.7万次

![情感分布图](./images/sentiment_trend_20251215_134236.png)

## 2. 舆情事件发展脉络

### 2.1 事件起源与发酵阶段
根据sentiment_time_series数据分析，事件可追溯至2024年6月初，早期仅有零星讨论。**关键转折点出现在2024年11月2日**，当日发帖量开始显著上升，夜骑开封相关话题开始形成规模效应。

### 2.2 高峰期（11月8日-11月9日）
- **2024年11月8日22:00**：发帖量首次突破370条，情感值达到3.17
- **2024年11月9日21:00**：达到舆情峰值507条，但情感值下降至2.84，负面情绪占比上升至30.18%
- **异常波动**：根据sentiment_anomaly_detection数据，11月8日-9日期间连续出现发帖量异常，偏离平均值2.8-7.9个标准差

### 2.3 缓和与收尾阶段（11月10日-11月30日）
舆情在11月10日后逐步降温，发帖量回归正常水平。根据topic_evolution数据，讨论焦点从单纯的骑行活动转向安全管理、交通管制等深层议题。

## 3. 舆情总体趋势分析

### 3.1 情感演化趋势
基于sentiment_time_series的hourly_data分析，整体情感呈现波动下降趋势：
- **初期（6月-10月）**：情感均值维持在3.0-4.0区间，以中性偏正面为主
- **高峰期（11月初）**：情感值显著波动，11月8日出现明显负面拐点
- **后期（11月中下旬）**：情感值逐步稳定在2.8-3.2区间

### 3.2 主题演化特征
topic_evolution数据显示主题关注度呈现"爆发式增长"特征：
- **民众反应**：从初期平均2.0条/日激增至峰值1,866条/日（11月9日）
- **交通运输**：从1.44条/日增长至1,523条/日（11月9日）

## 9. 舆情风险研判

### 9.1 风险等级评估
综合sentiment_anomaly_detection、volume_spikes和attribute_turning_points数据，当前舆情风险等级为**中等偏高**，主要风险点包括：

- **传播风险**：11月8-9日发帖量连续异常（Z-score 5.64-7.85），存在大规模传播风险
- **情感风险**：11月8-9日负面情绪占比升至30.18%，不满情绪激增130条
- **管理风险**：担忧情绪11月9日激增119条，显示公众对安全管理措施的担忧

### 9.2 风险预警信号
根据sentiment_time_series的turning_points数据，识别出以下关键预警信号：
- **情感拐点**：11月8日20:00，情感值从3.18降至2.85，负面情绪开始主导
- **主题转向**：11月8日，讨论焦点从"民众反应"转向"交通运输"和"政府工作"
- **发布者分化**：个人用户情感持续下降，而事业单位情感保持相对稳定

### 9.3 潜在风险场景
基于cross_dimension_matrix和relation_analysis的关联性分析，可能的风险场景包括：
- **群体对立**：不同发布者类型间情感差异可能加剧观点分化
